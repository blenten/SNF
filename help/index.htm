Ввод функций

 
 
В развёрнутом виде:

x1*x2*x3+x1*!x2*x3+!x1*!x2*x3

(x1+x2+x3)(x1+!x2+x3)(!x1+!x2+x3)

Переменные в конституенты вводятся в порядке возрастания индексов. При вводе СДНФ требуется явно обозначить конъюнкции при помощи знаков умножения.
При вводе СКНФ обозначать конъюнкции явно не обязательно. Для обозначения отрицания нужно поставить "!" перед переменной.

 
В числовой форме:

+(7,3,1)

*(7,3,1)

 Конституенты вводятся в скобки, через запятую.
Для обозначения типа СНФ, требуется поставить перед скобками "+" для СДНФ или "*" для СКНФ.
